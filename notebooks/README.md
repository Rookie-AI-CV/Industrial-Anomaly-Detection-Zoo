# Jupyter Notebooks

本目录包含解释性的Jupyter notebooks，用于演示和理解各种异常检测方法。

## 目录结构

- `reconstruction/`: 基于重建方法的notebooks
- `generation/`: 生成方法的notebooks
- `classification/`: 分类方法的notebooks
- `detection/`: 检测方法的notebooks
- `segmentation/`: 分割方法的notebooks

## 使用说明

1. 确保已安装Jupyter: `pip install jupyter`
2. 启动Jupyter: `jupyter notebook` 或 `jupyter lab`
3. 打开相应的notebook文件

## 建议格式

每个notebook应包含：
- 方法介绍
- 代码实现
- 可视化结果
- 结果分析
