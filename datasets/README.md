# 数据集目录

本目录用于存放工业异常检测相关的数据集。

## 常用数据集

### MVTec AD
- 官方网站：https://www.mvtec.com/company/research/datasets/mvtec-ad
- 描述：包含15个不同类别的工业异常检测数据集

### MVTec LOCO AD
- 官方网站：https://www.mvtec.com/company/research/datasets/mvtec-loco
- 描述：逻辑异常检测数据集

### CIFAR-10
- 官方网站：https://www.cs.toronto.edu/~kriz/cifar.html
- 描述：常用于异常检测基准测试

## 使用说明

1. 下载数据集到本目录
2. 按照数据集名称创建子目录
3. 在子目录中放置数据文件

## 注意事项

- 大型数据集文件不会被提交到git仓库
- 请参考各数据集的许可证要求
- 建议在README中记录数据集的下载链接和使用说明
