# 论文目录

本目录用于存放与工业异常和缺陷检测相关的论文和文献。

## 目录结构

```
papers/
├── survey/                    # 综述论文
├── reconstruction/            # 基于重建的方法论文
│   └── references/            # 引用论文
├── generation/                # 生成方法论文
│   └── references/            # 引用论文
├── classification/            # 分类方法论文
│   └── references/            # 引用论文
├── detection/                 # 检测方法论文
│   └── references/            # 引用论文
└── segmentation/              # 分割方法论文
    └── references/            # 引用论文
```

## 说明

- **综述论文** (`survey/`): 存放领域内的综述和调研论文
- **方法论文**: 按方法分类存放主要研究论文
- **引用论文** (`references/`): 每个方法类别下都有引用论文目录，用于存放相关论文中引用的其他文献，有助于深入理解方法的理论基础和相关工作

## 格式建议

- PDF文件命名：`[年份]_[第一作者]_[会议/期刊]_[标题].pdf`
- 例如：`2021_Ruff_ICML_Deep_One-Class_Classification.pdf`
